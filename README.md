# TFT-lcd-paint-program
advanced paint program for 2.4" tft lcd panel shields using the Arduino gfx library. Designed for arduino UNO R4 and will work on specific displays.
This code is for displays which are too slow and designed for the arduino uno R3.
Remember, if this code does not work on your display please change the code and tweak it to your liking.
Also you would need to change the touch calibration mappings depending on your display.
